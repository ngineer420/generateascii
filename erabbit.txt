<article class="prose">
  <h1>Project: generateascii.com</h1>

  A free ASCII art generator with text-to-banner and image-to-ASCII tools

  <h2>Stack</h2>

  <ul>
    <li>- HTML/CSS/JS: Vanilla, no framework or build step</li>
    <li>- Fonts: figlet.js, vendored with a curated 59-font catalogue</li>
    <li>- DevOps: Static files on GitHub Pages, custom domain via Cloudflare DNS</li>
  </ul>

  <h2>Objective</h2>

  To give one page two ASCII tools: FIGlet-style text banners with a
  browsable font gallery and color options, and drag-and-drop image-to-ASCII
  conversion via canvas luminance mapping with adjustable width, character
  ramp, and tone controls.

  <h2>Benefits</h2>

  <ul>
    <li>- Runs entirely client-side; uploaded images never leave the browser</li>
    <li>- 59 fonts and a live gallery make picking a style fast</li>
    <li>- .txt/.png export for both tools</li>
    <li>- No backend or database, so the whole site is free to host</li>
  </ul>

  <h2>Drawbacks</h2>

  <ul>
    <li>- Font list is a curated subset of the full FIGlet catalogue, not exhaustive</li>
  </ul>

  <h2>Easter Eggs</h2>

  A tiny rabbit sits tucked into the very bottom-right corner of the page,
  a nod to eRabb.it (erabb.it) — the site this one was built alongside.
